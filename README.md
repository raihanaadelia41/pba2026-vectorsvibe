# pba2026-vectorsvibe
Repository ini berfokus pada klasifikasi cyberbullying berbahasa Indonesia menggunakan Naive Bayes, Logistic Regression, dan SVM. Data diproses dengan teknik NLP dan TF-IDF. Tujuannya membandingkan performa model untuk menentukan metode terbaik dalam mendeteksi teks bully secara akurat.
